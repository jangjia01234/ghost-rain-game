# Demo

- 좌상단의 타이머를 통해 플레이 시간을 확인할 수 있습니다.
- 우상단의 볼륨 버튼을 통해 배경음악을 켜고 끌 수 있습니다.
- 키보드의 왼쪽과 오른쪽 키를 사용해 용사를 움직이며, 유령과 접촉하여 죽이는 게임입니다.
