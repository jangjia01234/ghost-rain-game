# Ghost Rain game 🎮
- 배포 링크: https://ghost-rain-game.vercel.app/

# Demo

- 키보드의 왼쪽 ⬅️ 과 오른쪽 ➡️ 키를 사용해 용사를 움직이며, 유령과 접촉하여 죽이는 게임입니다.
- 좌상단의 타이머 ⏰ 를 통해 남은 시간을 확인할 수 있습니다.
- 우상단의 볼륨 버튼을 통해 배경음악 🎸 을 켜고 끌 수 있습니다.
